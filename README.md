# 2024年全国大学生数学建模竞赛 C题：农作物种植策略优化系统

本项目针对2024年国赛C题“农作物的种植策略”构建了一套完整的数学建模与仿真求解方案。系统涵盖了从原始数据解析、混合整数线性规划（MILP）模型构建、蒙特卡洛风险压力测试到多因子关联动态均衡分析的全流程。

## 📌 项目背景

本研究立足于华北山区某乡村的农业规划，旨在有限的耕地资源、复杂的时序约束（如禁止重茬、三年一豆轮作）以及波动的市场环境下，寻求最优的种植方案以平衡经济效益与生态可持续性。

## 📁 文件结构

### 1\. 数据驱动层

  * **`data_loader.py`**：数据清洗与预处理引擎。负责解析附件中的地块物理属性、作物匹配矩阵及2023年历史种植状态，计算初始销售上限。

### 2\. 建模求解层

  * **`model_v2.py`**：针对问题 1.1 构建的基础优化模型。基于混合整数线性规划（MILP），实现了物理边界、季节隔离及三年轮作窗口等刚性约束。
  * **`model_v3.py`**：针对问题 1.2 的弹性销售模型。通过分段线性化处理，引入 50% 降价倾销逻辑，优化系统总利润。
  * **`model_v4.py`**：针对问题 2 的风险稳健模型。结合马科维茨风险惩罚项（Mean-Variance）调整预期单价，并集成 10,000 次蒙特卡洛模拟进行压力测试。
  * **`model_v5.py`**：针对问题 3 的市场均衡模型。采用序列线性规划（SLP）算法，通过多轮迭代模拟作物间的价格弹性与成本规模效应。

### 3\. 可视化与评估层

  * **`generate_pic1.py`**：宏观结构演变图、微观轮作热力图及 Top10 核心作物分布图生成。
  * **`generate_pic2.py`**：高级论文配图。包含真实数据填充的矩形树图、模型多维评价雷达图及风险-收益有效边界图。
  * **`sensitivity_analysis.py`**：针对价格、成本、需求上限等关键因子的控制变量灵敏度分析。

## 🧠 核心模型逻辑

1.  **约束体系**：

      * **时序约束**：相邻两年严禁重茬种植。
      * **生态约束**：每个地块在每连续三个年度内至少种植一次豆类作物（固氮轮作）。
      * **物理防线**：根据地块类型（平旱地、梯田、水浇地等）严格限制种植季节。

2.  **不确定性处理**：

      * **风险度量**：利用 $E(P) - \lambda \cdot \sigma$ 调整目标函数，对抗产量与单价的随机波动。
      * **稳定性评估**：通过蒙特卡洛模拟输出 95% 置信度下的 VaR（风险价值）底线利润。

## 🚀 快速上手

1.  **数据准备**：确保 `附件1.xlsx` 和 `附件2.xlsx` 放置在项目根目录。
2.  **执行求解**：
    ```bash
    python model_v2.py   # 求解 1.1 问并生成 result1_1_求解输出.xlsx
    python model_v3.py   # 求解 1.2 问并生成 result1_2_降价输出.xlsx
    python model_v4.py   # 执行风险模拟与压力测试
    ```
3.  **生成图表**：
    ```bash
    python generate_pic1.py
    python generate_pic2.py
    ```
